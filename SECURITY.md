# Security Policy | Politica de Seguranca

## Portugues

### Reportando uma Vulnerabilidade

Se voce descobrir uma vulnerabilidade de seguranca em qualquer repositorio desta organizacao, por favor reporte de forma responsavel.

Nao abra uma issue publica para problemas relacionados a seguranca.

Em vez disso, entre em contato de forma privada com os mantenedores do repositorio ou com os administradores da organizacao, incluindo:

- uma descricao clara da vulnerabilidade
- passos para reproduzir o problema
- o impacto potencial
- qualquer sugestao de correcao ou mitigacao, se houver

### Divulgacao Responsavel

Depois de receber um relato, os mantenedores avaliarao o problema e trabalharao em uma resposta adequada.

Por favor, aguarde um prazo razoavel para que a falha seja investigada e corrigida antes de compartilhar detalhes publicamente.

Agradecemos a divulgacao responsavel, que ajuda a proteger a comunidade e seus projetos.

## English

### Reporting a Vulnerability

If you discover a security vulnerability in any repository in this organization, please report it responsibly.

Please do not open a public issue for security-related problems.

Instead, contact the repository maintainers or organization administrators privately with:

- a clear description of the vulnerability
- steps to reproduce the issue
- the potential impact
- any suggested fix or mitigation, if available

### Responsible Disclosure

After receiving a report, maintainers will review the issue and work on an appropriate response.

Please allow reasonable time for the issue to be investigated and addressed before sharing details publicly.

We appreciate responsible disclosure that helps protect the community and its projects.
