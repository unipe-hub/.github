# Security Policy

### Reporting a Vulnerability

If you discover a security vulnerability in any repository in this organization, please report it responsibly.

Please do not open a public issue for security-related problems.

Instead, contact the repository maintainers or organization administrators privately with:

- a clear description of the vulnerability
- steps to reproduce the issue
- the potential impact
- any suggested fix or mitigation, if available

### Responsible Disclosure

After receiving a report, maintainers will review the issue and work on an appropriate response.

Please allow reasonable time for the issue to be investigated and addressed before sharing details publicly.

We appreciate responsible disclosure that helps protect the community and its projects.
